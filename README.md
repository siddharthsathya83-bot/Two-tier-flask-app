# Two-Tier Flask Application
A full-stack web application with automated CI/CD pipeline.

## 🛠️ Tech Stack
- **Frontend/Backend:** Python Flask
- **Database:** MySQL
- **Containerization:** Docker & Docker Compose
- **CI/CD:** Jenkins
- **Cloud:** AWS EC2

## 🚀 Features
- Add and view messages in real time
- Persistent data storage with MySQL
- Fully containerized with Docker
- Automated deployment with Jenkins pipeline
- GitHub Webhook for auto-triggering builds

## ⚙️ How to Run
git clone https://github.com/siddharthsathya83-bot/Two-tier-flask-app.git
cd Two-tier-flask-app
docker compose up -d --build
